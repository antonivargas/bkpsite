Introdução ao Aplicativo GeoSeller

Objetivo: Desenvolver uma ferramenta que venha a se tornar essencial para a estratégia de marketing, vendas e relacionamento com o cliente no mercado de varejo. Fazendo uso das tecnologias mobile explorando principalmente os recursos de geolocalização.

Definição: Aplicativo de ofertas para clientes com base na geolocalização dos respectivos usuários.

Aplicabilidade: Ferramenta voltada à marketing de gestão, elaboração e manipulação de dados para análise em gestão estratégica de negócios (prospecção, vendas e propaganda).
Palavras-chaves: Marketing, gestão e negócios.

Público Alvo: Mercado de varejo, lojas de departamentos, lojas especializadas, lojas de conveniência, varejo de serviços. Pequenos e médios negócios tanto no setor de comércio quanto serviços dos mais diversos segmentos.

Concorrência: Existem aplicações já consolidadas e também algumas que estão entrando no mercado que se encaixam como concorrentes diretos e indiretos de nossa solução. Uolet (Direto): Cria campanhas online com cupons de desconto e entrega através de geolocalização ainda em fase beta aplicativo já disponível para iphone e venda do serviço e criação de conta para estabelecimentos ainda não é feita pelo portal da Uolet.
Meu Carrinho (Indireto): Cria listas de compras e compara preços de estabelecimentos que vendem online.
FourSquare (Indireto): Vende publicidade online, com base na interação do usuário com o estabelecimento usando o sistema de geolocalização do foursquare.

Requisitos básicos:

- Portal Web para estabelecimentos comprarem seus planos de uso do serviço, e criarem suas campanhas promocionais.

- APP Mobile gratuito com uso de geolocalização para identificar a posição dos usuários e notificá-los sobre ofertas de estabelecimentos, á um raio de distância, que deverá ser definido pelo próprio usuário (Ferramenta semelhante ao filtro de raio do Tinder App) para receber ofertas cadastradas pelos estabelecimentos através do portal web.

Funcionamento

Estabelecimento

- Funcionamento das ofertas por parte do estabelecimento.

As ofertas criadas pelo estabelecimento serão feitas através de uma conta do mesmo criada no portal web www.geoseller.com | www.geoseller.com.br onde o estabelecimento cadastra uma oferta de um produto ou serviço gerando um voucher promocional, o estabelecimento define a quantidade limite que pretende atender, e o tempo de vigência de distribuição dos vouchers.

Usuário

- Funcionamento das ofertas por parte do usuário.

As ofertas para o usuário funcionarão através do recebimento de um código promocional junto das informações da referida oferta. O usuário comparece ao estabelecimento com o código promocional, o estabelecimento valida o código em sua conta durante a compra registrando que o usuário usufruiu da referida oferta. Desta forma também o estabelecimento tem o controle dos vouchers distribuídos e validados, dando início ao seu banco de dados de informações para ações de relacionamento com este perfil de cliente usuário do aplicativo. Quando o usuário receber uma notificação, para o voucher ser atribuído a sua conta pessoal, antes o mesmo deverá confirmar o recebimento do voucher, esta confirmação funcionará como um aceite por parte do usuário para contabilizar na conta do estabelecimento como voucher entregue.

Monetização

Monetizando o aplicativo:

- O estabelecimento compra um volume de vouchers para emitir via plataforma, pagando um valor X por voucher, podendo esta tarifa mudar para x-y de acordo com cada faixa de quantidade de vouchers que o estabelecimento adquirir (Quanto maior a quantidade, menor o valor unitário por voucher), a venda será feita sempre em pacotes/planos haverá um plano inicial free que será uma estratégia de captação de clientes através de uma experimentação da plataforma para o cliente conhecer o potencial da ferramenta, com uma pequena quantidade de cupons gratuitos podendo fazer upgrade para os demais planos que poderão ser adquiridos somente pelo portal web www.geoseller.com | www.geoseller.com.br assim que o cliente desejar contratar um novo plano. O sistema deverá permitir que o estabelecimento emita novamente em outras campanhas os vouchers já emitidos em uma campanha anterior mas que não foram entregues devido a não aceitação por parte do usuário.

Segurança da Forma de Monetização

- Controle dos Vouchers:

O sistema fará controle dos vouchers, gerando relatórios para análise de conversão referente a quantidade de vouchers entregues a usuários via geolocalização, vouchers entregues e validados, não validados, e vouchers do pacote do estabelecimento que não foram entregues devido a não aceitação do usuário, e os que não foram entregues devido a não emissão pelo estabelecimento.

- Assegurando o modelo financeiro:

O estabelecimento deverá pagar por todos seus vouchers, para evitar que o mesmo traga os clientes até seu estabelecimento via divulgação da oferta na plataforma, e efetive a venda promocional sem validar os vouchers recebidos pelos usuários. Ficando assim a validação somente como uma ferramenta de relatórios e estatísticas para o próprio estabelecimento analisar os resultados de conversão de suas campanhas.

Análise de mercado identificando o uso da web como apoio às vendas e estratégias de marketing no varejo, abaixo algumas aplicações usadas como objeto de análise e referências para embasar o escopo deste projeto.

- Groupon (Ofertas a partir de vouchers promocionais).
